# conscience
A full-stack productivity tool that helps you manage projects and tasks while providing motivational support to keep you on track. Unlike typical productivity trackers, **Conscience** blends project tracking with satire.

---

## Features
- Create and organise projects
- Break down projects into actionable tasks
- Visual progress tracking
- Log daily behaviours and habits
- Secure access to your data

## Tech Stack

- Frontend: React 18 + TypeScript + Vite
- Backend: Python FastAPI
- Database: SQLite with SQLAlchemy
- API: RESTful endpoints

## Quick Start

```bash
npm install
npm run dev:full
